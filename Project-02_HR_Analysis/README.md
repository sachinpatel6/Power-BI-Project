# HR Data Analysis Dashboard (Power BI)

## Project Overview
This project focuses on analyzing HR data to understand recruitment trends,
employee attrition, hiring cost, department-wise performance, and employee
joining patterns.

## Key Metrics
- Total Candidates
- Average Hiring Cost
- Time to Hire (TAT)
- Attrition within 90 days

## Key Insights
- Operations department has the highest hiring count
- Recruitment cost varies by hiring source
- Early attrition is more common in specific departments
- Performance score impacts employee retention

## Dashboard Features
- KPI cards for HR metrics
- Bar charts for department-wise hiring
- Funnel chart for recruitment stages
- Line chart for monthly recruitment trends
- Pie charts for department distribution

## Tools Used
- Power BI
- HR Dataset (Excel / CSV)
