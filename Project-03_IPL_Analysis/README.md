# IPL Data Analysis Dashboard (Power BI)

## Project Overview
This project analyzes IPL match data to understand team performance,
match outcomes, toss impact, venue analysis, and season-wise trends.

## Key Metrics
- Total Matches
- Total Wins
- Toss Wins
- Top Performing Players

## Key Insights
- Toss decision has a strong impact on match results
- Some teams consistently perform better across seasons
- Certain venues favor specific teams
- Player performance varies significantly by season

## Dashboard Features
- KPI cards for total matches and wins
- Bar charts for team-wise wins
- Line charts for season-wise performance
- Pie charts for toss win vs match win analysis
- Venue-based performance analysis

## Tools Used
- Power BI
- IPL Match Dataset
