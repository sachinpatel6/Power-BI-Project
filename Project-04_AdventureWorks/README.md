# AdventureWorks Sales Analysis Dashboard (Power BI)

## Project Overview
This project uses the AdventureWorks dataset to analyze sales, profit,
orders, customers, and product categories across different years and regions.

## Key Metrics
- Total Sales
- Total Profit
- Total Orders
- Profit Margin

## Key Insights
- Bikes category contributes the highest revenue
- Sales increased from 2015 to 2017
- North America generates the highest profit margin
- Customer segmentation helps identify high-value customers

## Dashboard Features
- KPI cards for business overview
- Tree map for category-level analysis
- Line charts for yearly sales and profit trends
- Pie charts for customer segmentation
- Region and category slicers

## Tools Used
- Power BI
- AdventureWorks Dataset
