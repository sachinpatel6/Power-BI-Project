# Power BI Projects P

This repository contains 4 Power BI dashboards created using real-world datasets.
Each project focuses on data analysis, data modeling, DAX calculations, and
interactive dashboard design to generate meaningful business insights.

These dashboards are built to show practical use of Power BI for business,
sports analytics, HR analytics, and sales analysis.

## Projects Included

### 1. Sales Data Analysis
A sales performance dashboard that analyzes sales, profit, discount, quantity,
customer segments, and regional trends.

### 2. HR Data Analysis
An HR analytics dashboard that analyzes recruitment, attrition, cost,
department-wise hiring, and performance scores.

### 3. IPL Data Analysis
An IPL analytics dashboard that focuses on team performance, wins, toss impact,
venues, seasons, and top players.

### 4. AdventureWorks Sales Analysis
An advanced sales dashboard using AdventureWorks data to analyze profit,
orders, customers, categories, and yearly trends.

## Tools Used
- Power BI
- Excel / CSV datasets
- Data Modeling
- DAX Measures
- Interactive Visualizations
