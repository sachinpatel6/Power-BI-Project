# Sales Data Analysis Dashboard (Power BI)

## Project Overview
This project focuses on analyzing sales data to understand overall business
performance. The dashboard highlights sales, profit, discount, quantity,
customer segments, and regional sales trends.

## Key Metrics
- Total Sales
- Total Profit
- Total Quantity Sold
- Total Discount

## Key Insights
- Sales performance varies significantly across states
- Certain products contribute more to profit than others
- Consumer segment generates the highest sales
- Sales and profit show noticeable trends over different years

## Dashboard Features
- KPI cards for quick business overview
- Bar charts for product and state-wise analysis
- Line chart for yearly sales and profit trends
- Donut and pie charts for category and segment analysis
- Interactive slicers for better filtering

## Tools Used
- Power BI
- Sales Dataset (CSV)
